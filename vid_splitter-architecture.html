vid_splitter — Smart Branching

Context-aware Jellyfin playback · Mature content filtering via BVF container format

Analysis Pipeline

  • • Whisper transcribes audio with word-level timestamps
  • • FFmpeg extracts 1 frame every 5 seconds
  • • Stable Diffusion Safety Checker classifies NSFW
  • • Detections merged into contiguous, keyframe-aligned segments
  • • Gap filling ensures full video coverage
  • • Outputs .bvf containers

Playback Architecture

  • • Plugin scans Jellyfin library on startup for manifests
  • • Registers virtual "Smart Branch" media sources
  • • ProfileResolver maps users → child/teen/adult
  • • SegmentServer swaps/skips mature content per profile
  • • Filler videos replace flagged segments
  • • Runs entirely inside Jellyfin — no external services

BVF Container Format

  • • Single self-contained file: manifest + all segments
  • • 64-byte header → 40-byte index entries → zstd manifest → data blocks
  • • O(1) random access via byte-offset index
  • • Profile-aware: multiple viewer profiles in one file
  • • Keyframe-aligned boundaries for seamless seeking
  • • Streamable: index at front, playback before full download

Profile System

  • • child (under 13): filters nudity, violence, language, fear
  • • teen_m (13–17, male): filters nudity, gore
  • • teen_f (13–17, female): filters nudity, violence
  • • adult (18+): no filters
  • • Auto-resolved from birthday + sex in plugin config
  • • ProfileOverride forces explicit profile assignment

Stack & Dependencies

  • • Python: whisper, diffusers, ffmpeg-python, torch, zstandard
  • • C#: .NET 9, Jellyfin SDK, System.Text.Json
  • • Models: Whisper (base/small/medium), SD Safety Checker
  • • GPU: Safety Checker can use CUDA for faster inference
  • • Codec support: H.264, H.265, AV1, VP9 + AAC, Opus, AC-3
